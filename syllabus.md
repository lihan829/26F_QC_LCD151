# Course Description

Students will learn foundational skills for working with natural language data and for processing text. These include basic Linux commands, an introduction to (Python) programming, with a focus on writing programs and applications related to text and language processing. Basic data structures and algorithms necessary for working with textual data: loops, recursion, hash tables. At the end of the semester, students will gradually build practical skills that prepare them for later courses in NLP, machine learning, and computational linguistics.

This course **does not** have any prerequisites or prior programming experience.

Students are expected to have regular access to a computer for completing programming assignments and participating in course activities. If you do not have a personal computer, you are encouraged to use the [Queens College Laptop Loan Program](https://www.qc.cuny.edu/library/visiting/computing/). Please contact the instructor as early as possible if access to a computer may be a barrier to your participation in the course.

---

# Learning Objectives

By the end of the course, students will be able to:
- Understand Python basics.
- Navigate Linux using the command line.
- Write simple programs to perform basic operations with text files
- Process structured linguistic data.
- Use lists, tuples, dictionaries, sets, and strings.
- Write functions and modular programs.
- Apply regular expressions to text processing.
- Analyze text corpora using Python.
- Use introductory NLP libraries including NLTK.

# Course Policies

## Exams

- Exams are intended to be taken **in person**.
- If Queens College changes its guidance and the course moves online, exams will be administered remotely.
- During online exams, students may be required to:
  - keep their camera on throughout the exam, and/or
  - complete a recorded oral follow-up to verify their understanding.
- Refusal to complete a required oral component will result in a grade of **0** for the exam.

### Make-up Exams

There are **no make-up exams**. Students who miss an exam without prior arrangements will receive a grade of **0**.

### Exam and Project Verification

If there is a significant discrepancy between a student's exam performance and project performance, the instructor may request an in-person or online oral meeting to discuss the submitted work. Final grades for the exam and/or project may be adjusted based on the student's demonstrated understanding during this meeting. Refusal to participate in the oral component will result in a grade of **0** for the affected assessment.

### Late Assignments

Assignments submitted after the deadline will incur a **20% penalty for each 24-hour period** they are late.

## Regrade Policy

If you believe an assignment or exam was graded incorrectly:

- **Exams:** Requests must be made during the **first office hour after the exam is returned**.
- **Assignments:** Requests must be made **within one week** of the graded assignment being returned.

Requests submitted after these deadlines will not be considered.
