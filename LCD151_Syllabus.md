# Course Description

This course introduces the computational tools and programming techniques that form the foundation of modern Computational Linguistics and Natural Language Processing (NLP). Students will learn Linux command-line skills, Python programming, and the essential data structures and algorithms needed to process linguistic data.

Emphasis is placed on writing programs that manipulate text, analyze corpora, and solve simple language-processing problems. Throughout the semester students will gradually build practical skills that prepare them for later courses in NLP, machine learning, and computational linguistics.

---

# Learning Objectives

By the end of the course, students will be able to:

- Navigate Linux using the command line.
- Use Git and GitHub for coursework.
- Develop programs in Python using VS Code and Jupyter Notebook.
- Read and write text files.
- Process structured linguistic data.
- Use lists, tuples, dictionaries, sets, and strings.
- Write functions and modular programs.
- Apply regular expressions to text processing.
- Analyze text corpora using Python.
- Use introductory NLP libraries including NLTK.
- Develop simple computational solutions to linguistic problems.
